This widget will generate an incoming call link for the Lifesize cloud service. You will need to provide your 7-digit Lifesize Video ID in the widget after you drop it into any menu.

== Description ==

The Lifesize plugin will allow you to conveniently post a "Call Me on Lifesize!" link to any menu in the Widget section of your Wordpress admin console.

Simply drag the Lifesize plugin to the desired menu, fill in your video ID, and hit save.

If the widget is provided nothing, no link will show up even if the widget is dragged into a menu.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Widget menu
4. Drag the Lifesize plugin to the desired sidebar or menu
5. Fill in your video ID
6. Hit save
